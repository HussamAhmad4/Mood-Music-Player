## Description  
**MoodSync** is a full-stack web application that dynamically generates Spotify playlists based on user mood and preferences. Unlike static playlist apps, MoodSync:
- Generates **fresh playlists** every time using Spotify API
- Features **user authentication** with personalized profiles
- Allows users to **save & share** mood playlists
- Provides **AI-enhanced mood detection** via user input
- Includes **social features** for community engagement
